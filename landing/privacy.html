WatchParty for Stremio

Privacy Policy

This policy applies to the WatchParty website at watchparty.mertd.me and the WatchParty for Stremio browser extension.

Last updated: 2026-04-23

What WatchParty does

WatchParty adds shared watch rooms to Stremio Web. It lets users create or join rooms, synchronize playback, chat during playback, share invite links, and manage room settings.

Data the extension and service use

  • Display name entered by the user for room participation.
  • Room data such as room ID, invite state, room visibility, bookmarks, reactions, and live chat messages.
  • Playback synchronization data such as play, pause, seek, timing, and connection state.
  • Stremio page context needed for the extension to identify the current title or active player route.
  • Local extension preferences such as accent color, compact chat, reaction sound, floating reactions, and selected backend mode.
  • Local Stremio runtime information from the user’s device, such as Stremio Server availability on localhost and basic server settings required for compatibility checks.
  • Stremio profile addon and settings data used to understand playback context. The extension keeps the Stremio auth key in extension memory only and does not persist it to extension storage.
  • GIF search queries sent by the user when using the GIF picker.

How data is used

  • To create, join, resume, and manage watch rooms.
  • To synchronize playback between room participants.
  • To deliver room chat, reactions, bookmarks, and host controls.
  • To detect local Stremio availability and support playback-related extension features.
  • To keep room state and user preferences available across popup, side panel, and in-page extension surfaces.

How data is shared

  • Room and synchronization data is sent to the WatchParty backend so shared rooms can function.
  • GIF search requests are sent to the GIF provider used by the extension when the user searches for GIFs.
  • WatchParty does not sell user data.
  • WatchParty does not use user data for advertising, credit decisions, or unrelated profiling.

Local storage

The extension stores room state, room keys, display name, and user preferences in Chrome extension storage so it can recover sessions and keep extension surfaces in sync. Sensitive Stremio auth state is kept in memory and is not persisted in extension storage.

Security

Private room keys are attached to invite links and used by the extension for private-room access. Local development and operational endpoints are restricted separately from the public production room flow.
