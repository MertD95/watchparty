How It Works

Get set up, invite people, and start syncing

The extension does the coordination work for you. The website is mainly for discovery, invites, and fast entry into rooms.
1

Install the extension

Add WatchParty to your browser, then make sure Stremio Web is open and ready.

2

Pick something to watch

Browse and start playing on web.stremio.com like normal.

3

Create or join a room

Use WatchParty to create or join a room, then let the extension carry the handoff into Stremio.

4

Watch in sync

Play, pause, seek, react, and chat while everyone stays aligned on the same timeline.
