Watch Together on Stremio

Synchronized playback with friends. Each person streams locally — only play/pause/seek events are synced. Free and open-source.

Install Extension Open Stremio Web

How It Works

1

Install the extension

Add WatchParty to your browser. Make sure Stremio desktop is running.

2

Pick something to watch

Browse and start playing on web.stremio.com as usual.

3

Create or join a room

Click the extension icon to create a room. Share the code with your friend.

4

Watch in sync

Play, pause, seek — it all syncs automatically. Chat in the sidebar overlay.
